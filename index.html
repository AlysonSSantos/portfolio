Projetos

A programação proporciona a criação de inúmeras ideias, aqui estão algumas das minhas:

Front-end

Landing-page para empresa dos EUA

Sendo meu primeiro trabalho como freelancer, formatei e melhorei uma landing-page para captação de leads de uma empresa de limpeza da cidade Boston. Com um toque de código, foi formatada, trazendo um design moderno e clean (sem trocadilhos), melhorada e recebeu novas funcionalidades. Este projeto está no ar e funcionando muito bem!

Lista de gastos com JS

Desenvolvido no período de um curso de JavaScript (Danki Code), a "Lista de compras" consiste em o usuário adicionar o nome e preço à produtos, podendo assim ter controle exatamente sobre o preço de cada um.

Back-end

ATM Simulator - C language

Sendo meu primeiro projeto independente de programação, o Automated Teller Machine é um programa em linguagem C, que busca simular um caixa eletrônico e algumas funções simples do mesmo (em breve no github).

Pedra, Papel ou tesoura

O clássico jogo de pedra, papel ou tesoura na web, onde o jogador enfrenta uma inteligência artificial capaz de escolher aleatoriamente (e muito bem!) entre pedra, papel ou tesoura. Proporcionando assim, um jogo disputado e com um placar dinâmico que conta o número de vitórias de cada um.
